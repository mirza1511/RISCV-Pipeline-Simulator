🚀 RISC-V Pipeline Simulator

A web-based **RISC-V 5-Stage Pipeline Simulator** developed as a **Semester Project** for the **Computer Organization & Assembly Language (COAL)** course during my **BS Artificial Intelligence** program at **Hamdard University**.

📖 Overview

This simulator provides an interactive visualization of the RISC-V 5-stage instruction pipeline, allowing users to understand how instructions flow through a processor while handling hazards and forwarding.

The project was built using **HTML**, **CSS**, and **JavaScript** with a focus on learning processor architecture through an intuitive graphical interface.

---

✨ Features

- ✅ 5-Stage RISC-V Pipeline
  - Instruction Fetch (IF)
  - Instruction Decode (ID)
  - Execute (EX)
  - Memory Access (MEM)
  - Write Back (WB)
- ✅ Data Hazard Detection (RAW)
- ✅ EX/MEM & MEM/WB Forwarding
- ✅ Load-Use Hazard Detection
- ✅ Branch Hazard Handling
- ✅ Pipeline Visualization
- ✅ Register File Monitoring
- ✅ Execution Statistics
  - Total Cycles
  - CPI
  - Stall Count
- ✅ Step-by-Step Simulation
- ✅ Automatic Pipeline Execution
